# 📜 Мои промпты для «Подарочного гуру»

Здесь я показываю, как формулировала запросы к ИИ, чтобы получить качественный контент для бота.

---

## Пример 1: Генерация идей подарков для категории «Мужчине»

**Задача:** получить список оригинальных идей подарков для мужчин с разным бюджетом.

**Техники:** `role prompting` (роль эксперта), `few-shot` (примеры), структурированный вывод.

**Промпт:**
Ты — эксперт по подаркам с 10-летним опытом. Вот несколько примеров хороших идей для мужчин:

• Умные часы с GPS → для активных, отслеживают маршруты и здоровье.

• Набор инструментов в кейсе → для домашнего мастера.

Теперь придумай 5 идей для мужчины 30–40 лет, который любит технологии и активный отдых. Для каждой идеи укажи:

• эмодзи

• краткое описание (не более 10 слов)

• примерную ценовую категорию (бюджетная - до 1000 рублей, средняя - до 5000 рублей, премиум - 5000 и более рублей)

**Результат (сокращённо):**
- ⌚ **Фитнес-браслет с измерением давления** – средняя
- 🎧 **Наушники с шумоподавлением** – средняя
- 🔋 **Пауэр банк с солнечной батареей** – бюджетная
- 🎒 **Непромокаемый рюкзак для ноутбука** – премиум
- 🥽 **Термобельё из мериноса** – средняя

## Пример 2: Генерация описания товара для карточки подарка

**Задача:** из названия и ключевых слов получить короткое продающее описание.

**Техники:** `zero-shot` с указанием формата, ограничение длины.

**Промпт:**

Напиши короткое описание (8–12 слов) для подарка «Умные часы с мониторингом здоровья».
Описание должно быть в настоящем времени, без слов «подойдёт», «поможет».
Начни с глагола или преимущества.


**Результат:**
> Отслеживают пульс, сон и калории. Мотивируют больше двигаться.

## Пример 3: Улучшение описания (итеративный промптинг)

**Задача:** первоначальное описание было слишком длинным и скучным. Я попросила ИИ переписать его в более динамичном стиле.

**Промпт:**
Перепиши описание: «Подойдут для отслеживания физической активности и контроля показателей здоровья — полезно для тех, кто ведёт активный образ жизни».

Сделай его короче (8–12 слов), убери «подойдут», «полезно». Добавь эмодзи в начало.


**Результат:**
> ⌚ Отслеживают пульс, сон и калории. Мотивируют больше двигаться.

---

## 🧠 Какие техники я использую

- **Role prompting** – задаю ИИ роль эксперта, чтобы повысить качество ответа.
- **Few-shot** – даю примеры перед задачей, чтобы показать желаемый формат и стиль.
- **Chain‑of‑thought** – разбиваю сложную задачу на шаги (например, сначала придумать названия, потом описания).
- **Zero‑shot с ограничениями** – явно указываю длину, тон, запрещённые слова.
